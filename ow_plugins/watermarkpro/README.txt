Instructions:

1.- Install the plugin with the plugin manager in your website.
2.- Click in configure button.
3.- Click on Image menu, upload your own logo or image.
4.- Click on Text menu and configure your text watermark.
    If you want to remove the text stroke, set the stroke size to 0
5.- Click on Preview menu, it will show you a preview of your settings
6.- Clic on Settings button and check the option "Start Watermarking" to enable the functionality for all the new uploaded pictures. You can enable only text watermark, only image watermark or both at the same time (<-default).
7.- Enjoy!

Important notes:
   The watermark can not be removed from a watermarked picture.
